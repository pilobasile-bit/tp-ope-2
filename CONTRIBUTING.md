# Contribuir al repositorio tp-ope-2

Este repositorio contiene el notebook `simulacion_tp2.ipynb` para la simulación del Trabajo Práctico 2 de Operativa.

## Cómo colaborar

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/pilobasile-bit/tp-ope-2.git
   ```
2. Crear una rama para tu trabajo:
   ```bash
   git checkout -b nombre-de-tu-rama
   ```
3. Trabajar en el notebook `simulacion_tp2.ipynb`.
4. Guardar los cambios y hacer commit:
   ```bash
   git add simulacion_tp2.ipynb
   git commit -m "Descripción del cambio"
   ```
5. Subir tu rama al repositorio remoto:
   ```bash
   git push origin nombre-de-tu-rama
   ```
6. Abrir un Pull Request en GitHub para revisar los cambios antes de mezclar en `master`.

## Reglas básicas

- No subas archivos personales o archivos binarios innecesarios.
- Usa `git status` para revisar qué archivos estás incluyendo.
- Si editas el notebook, asegúrate de guardar las celdas ejecutadas y revisar que el código corra.

## Contacto

Si necesitan ayuda, pueden escribirme o dejar un comentario en el Pull Request.
